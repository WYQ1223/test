git merge 分支合并、时空交融

查看用户名：git config user.name
修改用户名：git config --global user.name “修改的名字”
查看邮箱名：git config user.email
修改邮箱名：git config --global user.email “修改的邮箱名”
git add 追加（让系统承认文件）
git commit -am “init” 创建本地仓库，添加备注为init
git push 推到远程仓库
git branch 查看当前有几个分支
git checkout -b test1 创建一个名为test1的分支
git checkout 分支名 切换分支
git merge 文件名 分支合并、时空交融
git remote -v 查看详细远程分支
git status 查看当前状态
git log 查看版本信息
git reflog 查看历史纪录
git reset 版本回退
git diff 比较各版本区别
git push origin master 推到远程仓库

如果两个分支都对同一个文件做修改，在合并时很有可能会出现冲突
怎么解决冲突：手动进入文件修改顺序并删除一些内容，再退出后，输入git commit -am “merge”命令即可
